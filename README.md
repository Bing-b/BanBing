## 💡 项目简介

本项目基于 **VitePress** 构建，旨在打造一个极简、快速、且具备良好阅读体验的静态知识库。

- **极速响应**：基于 Vite 驱动，无论是开发环境还是生产环境，加载速度极快。
- **Markdown 驱动**：使用简单的 Markdown 语法即可快速生成美观的页面。

---

## 如何运行

- pnpm install
- pnpm run docs:dev

## 部署更新

- pnpm run docs:build
- sh ./vitepress-starter/deploy.sh
